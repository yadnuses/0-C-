## 一、语法细节（最容易丢分）

### 1. 中文标点是大坑
```c
printf("你好")；    // ❌ 中文分号
printf("你好");     // ✅ 英文分号
```
写代码前先切英文输入法。

### 2. 分号不能忘
```c
printf("请输入分数：\n")   // ❌ 缺分号
scanf("%d", &n)            // ❌ 缺分号
```
C 语言几乎每句结尾都要 `;`。

### 3. if / else 的正确写法
```c
if (n > 0); {          // ❌ if 后面不能有分号
if (n > 0) {           // ✅

} else (n < 0) {       // ❌ else 后面不能有条件
} else {               // ✅ 不需要条件
} else if (n < 0) {    // ✅ 如果要加条件，用 else if
```

## 二、核心概念（选择题高频）

### 4. `=` 和 `==` 的区别（问了两次，必须记住）
| 符号 | 作用 | 例子 |
|:---|:---|:---|
| `=` | 赋值（装进去） | `int a = 5;` |
| `==` | 比较（是否相等） | `if (a == 5)` |

口诀：一个等号是**装进去**，两个等号是**问一问**。

### 5. 0 是假，非 0 是真
```c
if (0)     // 假
if (1)     // 真
if (-5)    // 真
if (0.0)   // 假
```

### 6. `!=` 怎么打
```
Shift + 1  →  !
再按 =     →  !=
```
`!=` 是"不等于"，不是 `≠`。

## 三、运算符（计算题必考）

### 7. `/` 除法 vs `%` 取余
| 运算符 | 作用 | 例子 |
|:---|:---|:---|
| `/` | 商（能除几次） | `7 / 3 = 2` |
| `%` | 余数（剩多少） | `7 % 3 = 1` |

常见用法：`n % 2 == 0` 判断偶数。

### 8. 整数除法砍小数
```c
5 / 2     → 2      // 整数除整数，砍掉小数
5.0 / 2   → 2.5    // 有小数参与，保留小数
```

### 9. 浮点数不能直接用 `==` 比较
```c
0.1 + 0.2 == 0.3   // ❌ 结果是假（有精度误差）
fabs(0.1 + 0.2 - 0.3) < 1e-6   // ✅ 差值足够小就算相等
```

### 10. 科学计数法 `1e-6`
```
1e-6 = 1 × 10⁻⁶ = 0.000001
1e3  = 1 × 10³  = 1000
```
`e` 后面负数是小数，正数是大数。

## 四、switch 语句

### 11. break 不能忘
```c
case 1:
    printf("星期一\n");
    break;          // 没有 break 会穿透到 case 2
case 2:
    printf("星期二\n");
    break;
```

### 12. default 兜底
```c
default:
    printf("输入有误\n");
    break;
```
default 的提示要准确，不要写"大于3"但输入负数也走这里。

## 五、scanf 的对应关系

### 13. 格式字符串和变量一一对应
```c
scanf("%d %d", &a, &b);
//      │  │    │   │
//      │  │    │   └── 第二个值存进 b
//      │  │    └────── 第一个值存进 a
//      │  └─────────── 第二个数的格式
//      └────────────── 第一个数的格式
```

### 14. scanf 必须加 `&`
```c
scanf("%d", &n);    // ✅ 加 &
scanf("%d", n);     // ❌ 缺 &，程序崩溃
```
printf 不加 &，scanf 必须加 &。
